# 修复批量删除按钮显示问题

## 问题描述
在桌台管理页面没有看到批量删除按钮

## 问题原因
之前的逻辑是只有当selectedTableIds有值时才显示批量删除按钮，但这样用户永远无法进入选择模式（鸡生蛋蛋生鸡的问题）

## 解决方案
添加独立的批量选择模式：
1. 添加"批量删除"按钮，点击后进入批量选择模式
2. 在批量选择模式下，每个桌台显示选择框
3. 选择完成后显示"批量删除"和"退出选择"按钮
4. 删除完成后自动退出批量选择模式

## 修改文件
- 修改文件：frontend/src/views/TableManagement/index.vue

## 功能说明
1. 点击"批量删除"按钮进入批量选择模式
2. 在批量选择模式下，所有桌台卡片上显示选择框
3. 勾选要删除的桌台（只能删除空闲状态的桌台）
4. 点击"批量删除"按钮确认删除
5. 删除完成后自动退出批量选择模式
