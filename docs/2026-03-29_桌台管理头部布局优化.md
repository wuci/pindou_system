# 桌台管理头部布局优化

**日期：** 2026-03-29
**修改人：** wuci

## 修改内容

优化 `frontend/src/views/TableManagement/index.vue` 头部布局

### 调整前
- 标题独占一行
- 分类标签单独一行
- 统计信息单独一行
- 操作按钮独占右侧
- 存在滚动条问题

### 调整后
- 标题和操作按钮同行显示
- 分类标签和统计信息同行显示
- 减小间距：16px → 12px
- 减小字体：20px → 18px，14px → 13px
- 使用小尺寸按钮和标签
- 防止内容溢出

## 样式变更

```scss
.table-management {
  padding: 16px;  // 原 20px
  overflow: hidden;  // 新增：防止滚动条
}

.header-card :deep(.el-card__body) {
  padding: 16px;  // 默认值，确保紧凑
}

.header__title {
  font-size: 18px;  // 原 20px
}

.category-tab {
  padding: 5px 10px;  // 原 6px 12px
  font-size: 13px;  // 原 14px
  gap: 4px;  // 原 6px
}
```

## 目的
- 一眼看到所有内容
- 消除滚动条
- 提升视觉紧凑度
