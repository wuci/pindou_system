# 重构计费规则支持多渠道

## 需求描述
1. 支持多个支付渠道：店内、美团、大众点评
2. 每个渠道可以设置多条计费规则
3. 规则格式：X小时Y元，如1小时19元、2小时35元、4小时54元、不限时68元

## 数据结构设计

### 新的计费规则结构
```json
{
  "channels": [
    {
      "channel": "store",
      "channelName": "店内",
      "rules": [
        { "hours": 1, "price": 19, "unlimited": false },
        { "hours": 2, "price": 35, "unlimited": false },
        { "hours": 4, "price": 54, "unlimited": false },
        { "hours": null, "price": 68, "unlimited": true }
      ]
    },
    {
      "channel": "meituan",
      "channelName": "美团",
      "rules": [...]
    },
    {
      "channel": "dianping",
      "channelName": "大众点评",
      "rules": [...]
    }
  ]
}
```

## 修改文件

### 后端
1. **新建 DTO**:
   - `BillingRuleItemRequest.java` - 单条计费规则
   - `ChannelBillingRuleRequest.java` - 渠道计费规则
   - `BillingRuleConfigRequest.java` - 更新为包含渠道列表

2. **修改 Service**:
   - `ConfigServiceImpl.java` - 更新默认计费规则和保存逻辑

### 前端
1. **API类型定义** (`frontend/src/api/config.ts`):
   - 新增 `BillingRuleItem` 接口
   - 新增 `ChannelBillingRule` 接口
   - 更新 `BillingRule` 接口

2. **系统设置页面** (`frontend/src/views/Settings/index.vue`):
   - 新增渠道切换功能
   - 新增规则列表展示
   - 支持添加/删除规则
   - 支持设置不限时规则

## 功能说明
- 每个渠道独立配置计费规则
- 规则可以按时长排序设置
- 不限时规则表示封顶价格
- 系统自动匹配最优惠价格

## 默认配置
所有渠道默认规则：
- 1小时 19元
- 2小时 35元
- 4小时 54元
- 不限时 68元
