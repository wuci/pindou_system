## 修改内容
为"豆屿温柔集"设计并添加品牌Logo

### Logo设计理念

**设计元素**：
1. **豆子形状** - 椭圆形主体，略倾斜15°，象征"豆屿"
2. **渐变配色** - 淡紫→柔粉→淡蓝渐变，呼应温柔主题
3. **高光效果** - 增加立体感和温润质感
4. **小花装饰** - 三颗小圆点点缀，增添俏皮温柔感

**配色方案**：
- 主色：#d4a5ff（淡紫）
- 辅色：#ffb6d9（柔粉）
- 点缀：#c5e3ff（淡蓝）

### 应用位置

1. **登录页面** (`views/Login/index.vue`)
   - 大尺寸Logo：80×80px
   - 位置：标题上方
   - 动画：温柔浮动 + 轻微旋转

2. **系统头部** (`layouts/DefaultLayout.vue`)
   - 小尺寸Logo：36×36px
   - 位置：标题文字左侧
   - 动画：轻微上下浮动

### 动画效果

**登录页Logo**：
- 4秒循环
- 上下浮动 + 轻微旋转
- 柔和阴影

**头部Logo**：
- 3秒循环
- 轻微上下浮动
- 呼吸感动画
