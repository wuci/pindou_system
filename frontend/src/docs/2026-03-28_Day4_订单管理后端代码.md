## 技术要点

1. **计费逻辑**
   - 按小时计费：向上取整到小时
   - 按分钟计费：向上取整到分钟
   - 超时判断：实际时长 > 预设时长
   - 超时费用 = 超时时长 × 单价 × 超时费率

2. **金额精度**
   - 使用`Math.round(amount * 100.0) / 100.0`保留两位小数

3. **时间线记录**
   - 当前从订单数据构建
   - 后续可从操作日志表读取（sys_operation_log）

4. **容错机制**
   - 计费规则配置不存在时使用默认值
   - JSON解析失败时使用默认值
