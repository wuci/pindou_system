## 技术要点

1. **实时计费**
   - 账单接口实时计算当前费用
   - 支持暂停/恢复状态下的准确计费

2. **费用明细JSON**
   ```json
   {
     "normalAmount": 60.00,
     "overtimeAmount": 0.00,
     "totalAmount": 60.00,
     "actualDuration": 7200,
     "billingType": "hour",
     "unitPrice": 30.0,
     "overtimeRate": 1.5
   }
   ```

3. **结账流程**
   - 点击结束按钮 → 显示账单弹窗 → 确认结账 → 调用接口 → 刷新列表

4. **时长计算**
   - 总时长 = 当前时间 - 开始时间
   - 暂停时长 = 累计暂停 + 当前暂停（如正在暂停）
   - 计费时长 = 总时长 - 暂停时长
