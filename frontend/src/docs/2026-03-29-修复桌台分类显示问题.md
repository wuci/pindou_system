# 修复桌台分类显示问题

## 日期
2026-03-29

## 问题描述
用户反馈：点击桌台配置并保存后，界面上查不出来分类信息。

## 问题原因
后端 `TableInfoResponse` 类中缺少 `categoryId` 字段，导致前端无法获取桌台分类信息。

## 修复内容

### 1. 后端 - 添加 categoryId 字段到响应DTO
**文件：** `backend/pdsystem-entity/src/main/java/com/pindou/timer/dto/TableInfoResponse.java`

```java
@Schema(description = "分类ID")
private Long categoryId;
```

### 2. 后端 - 设置 categoryId 到响应
**文件：** `backend/pdsystem-service/src/main/java/com/pindou/timer/service/impl/TableServiceImpl.java`

在 `convertToResponse` 方法中添加：
```java
response.setCategoryId(table.getCategoryId());
```

## 测试验证
1. 重新编译后端：`mvn clean package -DskipTests`
2. 重启后端服务（端口 9026）
3. 前端可以正常获取和显示桌台分类信息

## 影响
- 桌台列表现在正确显示分类信息
- 分类筛选功能正常工作
- 编辑桌台分类功能正常工作
