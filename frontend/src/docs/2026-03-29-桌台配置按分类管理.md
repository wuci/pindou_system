# 桌台配置按分类管理

## 日期
2026-03-29

## 需求描述
1. 桌台配置只针对每个分类单独管理
2. 配置的桌台只属于对应的分类
3. 点击"全部"分类时，不显示桌台配置按钮

## 修改内容

### 后端修改

#### 1. TableConfigRequest 添加 categoryId 字段
**文件：** `backend/pdsystem-entity/src/main/java/com/pindou/timer/dto/TableConfigRequest.java`

```java
@Schema(description = "分类ID", required = true, example = "1")
@NotNull(message = "分类ID不能为空")
private Long categoryId;
```

#### 2. TableServiceImpl.configTableCount 方法重构
**文件：** `backend/pdsystem-service/src/main/java/com/pindou/timer/service/impl/TableServiceImpl.java`

修改逻辑：
- 按分类ID查询当前桌台数量
- 增加桌台时设置分类ID
- 减少桌台时只删除该分类下的空闲桌台

### 前端修改

#### 1. TableConfigDialog 添加分类选择
**文件：** `frontend/src/views/TableManagement/components/TableConfigDialog.vue`

- 添加分类选择下拉框
- 显示当前分类的桌台数量
- 传入 categoryId、categories、tables 作为 props

#### 2. 修改 API 调用
**文件：** `frontend/src/api/table.ts`

```typescript
export const configTableCount = (tableCount: number, categoryId: number) => {
  return http.put('/tables/config', { tableCount, categoryId })
}
```

#### 3. TableManagement 页面控制按钮显示
**文件：** `frontend/src/views/TableManagement/index.vue`

- 添加条件：当 currentCategory === 0 时隐藏桌台配置按钮
- 传递 categories、tables、categoryId 给 TableConfigDialog

## 使用说明
1. 选择具体分类（如"大厅"、"包间"）后，显示"桌台配置"按钮
2. 点击"桌台配置"，选择分类和数量
3. 点击"全部"分类时，不显示"桌台配置"按钮
