## 修改内容
创建订单详情抽屉组件（OrderDetailDrawer.vue）

## 问题分析
启动前端时报错：`ENOENT: no such file or directory, open 'E:\ai_project\pindou_system-main\frontend\src\components\OrderDetailDrawer.vue'`

原因：`OrderManagement/index.vue` 中引用了 `OrderDetailDrawer.vue` 组件，但该组件文件不存在。

## 解决方案
根据 Day 4 排期计划要求，创建订单详情抽屉组件，实现以下功能：

### 组件功能
1. **订单基本信息展示**
   - 桌台名称
   - 订单状态（进行中/已完成）
   - 操作员
   - 预设时长

2. **时间信息展示**
   - 开始时间
   - 结束时间（已完成订单）
   - 使用时长
   - 暂停时长
   - 计费时长
   - 支付时间

3. **费用明细展示**
   - 正常费用
   - 超时费用（如有）
   - 总费用

4. **操作记录时间线**
   - 使用 el-timeline 组件
   - 显示操作时间、动作、描述和操作人

### 技术实现
- 使用 Vue 3 Composition API
- 使用 Element Plus Drawer 组件
- 响应式数据加载
- 时间格式化、时长格式化、金额格式化工具函数
