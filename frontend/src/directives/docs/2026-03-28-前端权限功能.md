## 一、实现内容

### 1. v-permission 指令

- `directives/permission.ts` - 权限指令实现
- `directives/index.ts` - 指导出和注册

**使用方式：**
```vue
<!-- 单个权限 -->
<el-button v-permission="'user:create'">新增用户</el-button>

<!-- 多个权限（满足任意一个） -->
<el-button v-permission="['user:create', 'user:edit']">操作</el-button>
```

### 2. usePermission Hook

- `composables/usePermission.ts` - 权限检查Hook

**功能：**
- hasPermission() - 是否拥有权限（任意一个）
- hasAllPermissions() - 是否拥有所有权限
- hasAnyPermission() - 是否拥有任意权限
- permissions - 当前用户权限列表
- isSuperAdmin - 是否超级管理员
- isAdmin - 是否管理员
- role - 用户角色

**使用示例：**
```ts
import { usePermission } from '@/composables/usePermission'

const { hasPermission, hasAllPermissions } = usePermission()

// 检查单个权限
if (hasPermission('user:create')) {
  // ...
}

// 检查多个权限（全部）
if (hasAllPermissions(['user:create', 'user:edit'])) {
  // ...
}
```

## 二、权限说明

系统支持的权限代码：
- 用户管理: user:view, user:create, user:update, user:delete, user:resetPassword
- 角色管理: role:view, role:create, role:update, role:delete
- 桌台管理: table:view, table:create, table:update, table:delete
- 订单管理: order:view, order:create, order:update, order:delete, order:export
- 配置管理: config:view, config:update
- 统计报表: statistics:view
