## 组件概述
`RemindPanel.vue` 是一个固定在页面右侧的提醒通知面板组件，用于展示桌台时长提醒信息。

## 涉及文件

### 新增文件
1. `components/RemindPanel.vue` - 提醒面板组件
2. `api/remind.ts` - 提醒相关 API
3. `layouts/DefaultLayout.vue` - 主布局（已集成提醒面板）

## 组件特性

### 1. 可折叠设计
- 点击头部可折叠/展开面板
- 折叠状态下仅显示图标
- 展开状态下显示完整提醒列表

### 2. 提醒状态样式
- **即将到期**:
  - 黄色边框 (border-left-color: #e6a23c)
  - 渐变背景
  - 滑入动画 (slideIn)

- **已超时**:
  - 红色边框 (border-left-color: #f56c6c)
  - 脉冲动画 (pulse)
  - 持续闪烁提醒

### 3. 声音提示
- 支持开关控制
- 音量: 50%
- 音频文件: /sounds/remind.mp3

### 4. 自动刷新
- 间隔: 5秒
- 组件销毁时自动停止定时器

### 5. 交互功能
- 忽略单个提醒
- 全部已阅清空
- 父组件事件通知

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| soundEnabled | Boolean | true | 是否启用声音 |

## Events

| 事件 | 参数 | 说明 |
|------|------|------|
| remind | RemindInfo[] | 有新提醒时触发 |

## 暴露方法

| 方法 | 说明 |
|------|------|
| loadReminders | 手动加载提醒列表 |
| stopRefresh | 停止自动刷新 |

## 样式类

| 类名 | 说明 |
|------|------|
| .remind-panel | 面板容器 |
| .collapsed | 折叠状态 |
| .expiring | 即将到期样式 |
| .timeout | 超时样式 |

```vue
<script setup>
import RemindPanel from '@/components/RemindPanel.vue'

const handleRemind = (reminds) => {
  console.log('收到提醒:', reminds)
}
</script>
```

## API 说明

### getReminders()
获取需要提醒的桌台列表

```typescript
const data = await getReminders()
// 返回: RemindInfo[]
```

### ignoreRemind(tableId)
忽略指定桌台的提醒

```typescript
await ignoreRemind(tableId)
// 返回: boolean
```
