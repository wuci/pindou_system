# 拼豆计时管理系统 - 开发排期计划

## 一、排期概览

| 项目 | 内容 |
|------|------|
| **项目周期** | 15个工作日（3周） |
| **开发模式** | 前后端并行开发 |
| **团队规模** | 前端1人 + 后端1人 |
| **交付目标** | MVP版本，核心功能可用 |

---

## 二、开发优先级

| 优先级 | 模块 | 说明 |
|--------|------|------|
| **P0** | 认证授权、桌台管理、订单管理、计时计费 | 核心业务流程 |
| **P1** | 用户管理、角色管理、系统配置 | 管理功能 |
| **P2** | 数据统计、操作日志、提醒系统 | 增强功能 |

---

## 三、技术栈

### 前端技术栈
| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.4+ | 前端框架 |
| Vite | 5.x | 构建工具 |
| Element Plus | - | UI组件库 |
| Pinia | - | 状态管理 |
| Vue Router | 4.x | 路由管理 |
| Axios | - | HTTP客户端 |
| ECharts | - | 图表库 |
| TypeScript | - | 类型系统 |

### 后端技术栈
| 技术 | 版本 | 用途 |
|------|------|------|
| Spring Boot | 2.7.18 | 基础框架 |
| JDK | 1.8 | 运行环境 |
| MyBatis-Plus | 3.5.5 | ORM框架 |
| MySQL | 8.0+ | 关系数据库 |
| Redis | 6.x | 缓存 |
| Druid | 1.2.20 | 数据库连接池 |
| JWT | 0.11.5 | Token认证 |
| Knife4j | 4.1.0 | API文档 |
