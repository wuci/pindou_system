# 启动脚本修改记录

## 修改日期
2026-04-06

## 修改内容
1. 支持从外部目录读取静态资源文件
2. 修改JDK最低版本要求为 JDK 8

## 修改详情

### 1. 新增配置项
- `STATIC_RESOURCE_DIR="./static"` - 外部静态资源目录路径

### 2. 新增函数
- `check_static_resources()` - 检查并配置外部静态资源目录
  - 检查目录是否存在
  - 检查目录是否为空
  - 自动添加 Spring Boot 启动参数

### 3. 启动流程调整
- `start` 命令：新增静态资源检查
- `restart` 命令：新增静态资源检查
- 更新帮助信息，说明静态资源配置方式

### 4. 技术实现
使用 Spring Boot 参数指定外部静态资源位置：
```bash
--spring.web.resources.static-locations=file:./static
```

### 5. JDK版本检查优化
- 兼容 JDK 8 的版本格式（1.8.x）
- 兼容 JDK 9+ 的版本格式（x.x.x）
- 最低版本要求修改为 JDK 8

## 使用方法

1. 创建静态资源目录：
```bash
mkdir -p static
```

2. 将前端打包文件放入 static 目录

3. 启动服务：
```bash
./start.sh start
```

## 优先级
外部静态资源 > JAR包内置静态资源
