## Nacos 配置说明

### 命名空间
- ID: pd_system

### 分组
- pd_system

### Data ID
1. **common.yml** - 公共配置
   - 数据库配置
   - Redis配置
   - 通用参数

2. **pd_server.yml** - 服务特定配置
   - 服务端口
   - 业务配置
   - JWT配置

## 配置加载优先级
1. bootstrap.yml (Nacos配置中心连接信息)
2. Nacos 远程配置 (common.yml, pd_server.yml)
3. application.yml (本地配置，可覆盖Nacos配置)
4. 环境变量/命令行参数

## 使用说明
1. 确保Nacos服务正常运行
2. 在Nacos控制台创建对应的命名空间和配置
3. 配置格式选择 YAML
4. 应用启动时会自动从Nacos拉取配置

## 注意事项
- namespace 需要使用命名空间ID，不是名称
- 如需动态刷新配置，在类上添加 @RefreshScope 注解
- 配置修改后Nacos会自动推送到客户端
