## 功能概述
实现桌台使用时长提醒系统，支持即将到期提醒和超时提醒。

## 涉及文件

### 新增文件
1. `RemindTask.java` - 提醒定时任务
   - 每5秒检查一次提醒状态
   - 发送提醒通知（日志记录，可通过WebSocket扩展）

2. `controller/RemindController.java` - 提醒控制器
   - GET /remind - 获取提醒列表
   - POST /remind/ignore/{tableId} - 忽略提醒

### 修改文件
1. `RemindServiceImpl.java` - 提醒服务实现
   - 检查所有使用中的桌台
   - 判断是否需要提醒（即将到期/已超时）
   - 支持忽略提醒功能

2. `DefaultLayout.vue` - 主布局
   - 集成 RemindPanel 组件

## 功能特性

### 1. 提醒类型
- **即将到期**: 剩余时间 <= 配置阈值（默认5分钟）
- **已超时**: 已用时长 > 预设时长，支持重复提醒

### 2. 提醒配置
- threshold: 提前提醒阈值（秒），默认300秒（5分钟）
- repeatInterval: 超时重复提醒间隔（秒），默认60秒（1分钟）

### 3. 桌台状态字段
- reminded: 是否已发送即将到期提醒
- remindIgnored: 是否已忽略提醒
- lastPauseTime: 最后提醒时间（复用字段）

### 4. 前端提醒面板
- 可折叠侧边面板
- 声音提示
- 即将到期（黄色）/ 已超时（红色脉冲动画）
- 自动刷新（每5秒）
- 支持忽略单个/全部提醒

## 接口说明

### 获取提醒列表
```
GET /remind
Response: List<RemindInfo>
```

### 忽略提醒
```
POST /remind/ignore/{tableId}
Response: Boolean
```

## 数据结构

### RemindInfo
| 字段 | 类型 | 说明 |
|------|------|------|
| tableId | Long | 桌台ID |
| tableName | String | 桌台名称 |
| remindType | String | 提醒类型：expiring/timeout |
| remindTypeDesc | String | 提醒类型描述 |
| startTime | Long | 开始时间戳 |
| presetDuration | Integer | 预设时长（秒） |
| usedDuration | Integer | 已用时长（秒） |
| remainingDuration | Integer | 剩余时长（秒） |
| overtimeDuration | Integer | 超时时长（秒） |
