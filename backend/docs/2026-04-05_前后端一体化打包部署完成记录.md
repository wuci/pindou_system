### 3. NPM 依赖问题
**问题：** terser 未安装

**修复：** `npm install terser --save-dev`

### 4. Lombok 注解处理器问题
**问题：** JDK 1.8.0_131 与 Lombok 注解处理器不兼容

**修复：** 将所有 `@Slf4j` 注解替换为手动 Logger 声明

**修改文件数：** 24 个 Java 文件

**修改模板：**
```java
// 删除
import lombok.extern.slf4j.Slf4j;
@Slf4j

// 添加
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;
private static final Logger log = LoggerFactory.getLogger(ClassName.class);
```

### 5. Resource 类型冲突
**文件：** `WebMvcConfig.java`

**问题：** `javax.annotation.Resource` 注解与 `org.springframework.core.io.Resource` 类型冲突

**修复：** 使用全限定名 `org.springframework.core.io.Resource`

### 6. SPA 控制器文件读取问题
**文件：** `ETSSpaFallbackController.java`

**问题：** `Paths.get(resource.getURI())` 在 JAR 包中无法工作

**修复：** 使用 `resource.getFile().toPath()` 替代
