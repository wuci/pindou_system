# 修复 JAR 包中静态资源访问问题

## 日期
2026-04-05

## 任务概述
修复前后端一体化打包后，静态资源在 JAR 包中无法访问的问题，实现根路径访问前端应用。

## 完成状态
✅ **修复完成**
✅ **启动成功**
✅ **访问正常**

---

## 问题分析

### 原始问题
1. 应用 context-path 设置为 `/api`，导致根路径 `/` 不在 Spring Boot 处理范围内
2. ETSSpaFallbackController 使用 `resource.getFile().toPath()` 无法读取 JAR 包内的资源

### 解决方案
1. 移除 context-path 配置，将其设置为默认值 `/`
2. 为所有 API 控制器添加 `/api` 前缀
3. 修复 ETSSpaFallbackController 使用 InputStream 读取资源
4. 更新 SecurityConfig 和 WebMvcConfig 配置

## 修改的文件

### 1. application.yml
**文件：** `backend/pdsystem-server/src/main/resources/application.yml`

**修改：** 移除 context-path 配置
```yaml
# 修改前
server:
  port: 9026
  servlet:
    context-path: /api  # 移除此行

# 修改后
server:
  port: 9026
  servlet:
    encoding:
      charset: UTF-8
```

### 2. ETSSpaFallbackController.java
**文件：** `backend/pdsystem-server/src/main/java/com/pindou/timer/controller/ETSSpaFallbackController.java`

**修改：** 使用 InputStream 读取 JAR 包内的资源
```java
// 修改前
indexHtmlContent = new String(Files.readAllBytes(
        resource.getFile().toPath()), StandardCharsets.UTF_8);

// 修改后
try (BufferedReader reader = new BufferedReader(
        new InputStreamReader(resource.getInputStream(), StandardCharsets.UTF_8))) {
    indexHtmlContent = reader.lines().collect(Collectors.joining("\n"));
    spaEnabled = true;
}
```

### 3. ETSBaseController.java（新建）
**文件：** `backend/pdsystem-server/src/main/java/com/pindou/timer/controller/ETSBaseController.java`

**说明：** API 控制器基类，所有 API 控制器继承此类
```java
@RequestMapping("/api")
public abstract class ETSBaseController {
    // API 控制器基类，无需额外实现
}
```

### 4. 所有 API 控制器
**文件：** `backend/pdsystem-server/src/main/java/com/pindou/timer/controller/*.java`

**修改：** 在 @RequestMapping 中添加 `/api` 前缀
```java
// 修改前
@RequestMapping("/auth")
public class AuthController {

// 修改后
@RequestMapping("/api/auth")
public class AuthController extends ETSBaseController {
```

### 5. SecurityConfig.java
**文件：** `backend/pdsystem-server/src/main/java/com/pindou/timer/config/SecurityConfig.java`

**修改：** 更新安全配置，放行前端静态资源和更新后的 API 路径
```java
// 放行前端静态资源
.antMatchers(
    "/",
    "/index.html",
    "/assets/**",
    "/static/**",
    "/favicon.ico",
    "/login",
    "/dashboard/**",
    "/tables/**",
    "/orders/**",
    "/reports/**",
    "/users/**",
    "/roles/**",
    "/logs/**",
    "/members/**",
    "/member-levels/**",
    "/settings/**"
).permitAll()

// 放行公共接口（登录接口等）- 注意现在有 /api 前缀
.antMatchers("/api/auth/**").permitAll()
```

### 6. WebMvcConfig.java
**文件：** `backend/pdsystem-server/src/main/java/com/pindou/timer/config/WebMvcConfig.java`

**修改：** 更新拦截器配置，只拦截 `/api/**` 路径
```java
registry.addInterceptor(permissionInterceptor)
        .addPathPatterns("/api/**")
        .excludePathPatterns(
                "/api/auth/login",
                "/api/auth/captcha",
                "/api/health/**",
                "/error"
        );
```

### 7. StaticResourceConfig.java（删除）
**文件：** `backend/pdsystem-server/src/main/java/com/pindou/timer/config/StaticResourceConfig.java`

**说明：** 此文件在修改过程中创建后被删除，不需要单独的静态资源配置
