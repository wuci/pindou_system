## 数据库表结构（共12张表）

### 系统管理表（4张）

1. **sys_user** - 用户表
   - 用户ID、用户名、密码、昵称
   - 角色ID、状态、登录信息
   - 软删除支持

2. **sys_role** - 角色表
   - 角色ID、名称、编码
   - 权限列表（JSON格式）
   - 排序、状态、内置标识

3. **sys_operation_log** - 操作日志表
   - 操作用户、操作类型
   - 目标类型、目标ID
   - IP地址、执行时间

4. **sys_config** - 系统配置表
   - 配置键、配置值（JSON）
   - 配置描述、更新时间

### 业务表（8张）

5. **biz_member_level** - 会员等级配置表
   - 等级名称、金额范围
   - 折扣率、排序

6. **biz_member** - 会员表
   - 会员信息（姓名、手机号、地址）
   - 累计消费、余额
   - 会员等级ID

7. **biz_table_category** - 桌台分类表
   - 分类名称、图标
   - 排序、备注

8. **biz_table_layout_config** - 桌台布局配置表
   - 分类ID（0表示全局）
   - 布局配置（JSON）

9. **biz_table** - 桌台表
   - 桌台编号、名称
   - 分类ID、状态
   - 当前订单、计时信息
   - 预定信息
   - 提醒信息

10. **biz_order** - 订单表
    - 订单ID、桌台信息
    - 时间信息（开始、结束、时长）
    - 状态、支付方式
    - 金额信息（总金额、余额支付、其他支付）
    - 会员信息、折扣
    - 操作员、备注

11. **biz_recharge_record** - 充值记录表
    - 会员信息
    - 充值金额、前后余额
    - 支付方式
    - 操作员

12. **biz_consumption_record** - 消费记录表
    - 会员信息
    - 订单ID
    - 消费金额、前后余额
    - 备注

## 关键特性

### 1. 支持重复执行

**建表脚本：**
```sql
SET FOREIGN_KEY_CHECKS = 0;
DROP TABLE IF EXISTS `table_name`;
CREATE TABLE `table_name` ...
```

**初始化数据脚本：**
```sql
INSERT IGNORE INTO `table_name` ...
```

### 2. 完整的外键关系

```
sys_user.role_id → sys_role.id
biz_member.level_id → biz_member_level.id
biz_recharge_record.member_id → biz_member.id
biz_consumption_record.member_id → biz_member.id
```

### 3. 统一时间戳

所有表的时间字段统一使用：
- `created_at` BIGINT - 毫秒时间戳
- `updated_at` BIGINT - 毫秒时间戳

### 4. 软删除支持

系统管理表支持软删除：
- `deleted_at` BIGINT - 删除时间
