# 桌台管理添加滚动支持

## 日期
2026-04-04

## 问题描述
桌台管理页面当桌台数量较多超出屏幕时，无法滚动查看所有桌台。

## 修改内容

### 文件：frontend/src/components/TableLayoutEditor.vue

**1. 组件根容器样式**
```scss
.table-layout-editor {
  width: 100%;
  height: 100%;
  display: flex;
  flex-direction: column;
  overflow: hidden;
}
```

**2. 网格视图样式**
```scss
.grid-view {
  flex: 1;
  padding: 10px;
  overflow-y: auto;
  overflow-x: hidden;
}

.table-grid {
  padding-bottom: 20px;  // 底部额外空间
}
```

**3. 布局视图样式**
```scss
.layout-view {
  flex: 1;
  position: relative;
  min-height: 400px;
  overflow: hidden;
  display: flex;
  flex-direction: column;
}
```

**4. 布局画布样式**
```scss
.layout-canvas {
  position: relative;
  width: 100%;
  flex: 1;
  min-height: 100%;
  background: #ffffff;
  border-radius: 8px;
  overflow: auto;
  cursor: default;
  transition: all 0.3s;
  padding-bottom: 40px;  // 底部额外空间
}
```

## 修改说明

1. **使用 Flex 布局**：组件使用 `display: flex` 和 `flex-direction: column` 来正确分配空间

2. **网格视图滚动**：
   - 设置 `flex: 1` 占用剩余空间
   - 设置 `overflow-y: auto` 支持垂直滚动
   - 设置 `overflow-x: hidden` 隐藏水平滚动条

3. **布局视图滚动**：
   - 设置 `flex: 1` 占用剩余空间
   - 布局画布 `overflow: auto` 支持滚动
   - 设置 `min-height: 400px` 确保有最小高度

## 影响范围
- 桌台管理页面的网格视图和布局视图

## 测试建议
1. 添加多个桌台，验证网格视图可以滚动查看
2. 切换到布局视图，验证画布可以滚动查看
3. 在小屏幕下测试滚动功能
