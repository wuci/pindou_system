# 2026-04-06 订单折扣记录功能

## 修改描述
实现订单折扣记录和平均折扣率计算功能。支持记录多次活动折扣，并计算平均折扣率。

## 修改文件

### 1. Entity层
**文件**: `pdsystem-entity/src/main/java/com/pindou/timer/entity/Order.java`

**修改内容**:
- 新增 `discountIds` 字段：存储应用的折扣ID列表（JSON数组格式）
- 新增 `discountCount` 字段：记录应用的折扣次数
- 新增 `averageDiscountRate` 字段：存储多次折扣的平均折扣率
- 新增 `totalDiscountAmount` 字段：存储总折扣金额

### 2. Service层
**文件**: `pdsystem-service/src/main/java/com/pindou/timer/service/impl/TableServiceImpl.java`

**修改内容**:
1. 修改 `AmountCalculationResult` 内部类，添加折扣相关属性：
   - `discountId`: 折扣ID
   - `discountRate`: 折扣率

2. 修改 `ExtendCalculationResult` 内部类，添加折扣相关属性：
   - `discountId`: 新增的折扣ID
   - `discountRate`: 新增的折扣率
   - `newDiscountCount`: 更新后的折扣次数
   - `newAverageDiscountRate`: 更新后的平均折扣率
   - `newTotalDiscountAmount`: 更新后的总折扣金额

3. 新增 `ExtendDiscountCalculation` 内部类：用于封装续费折扣计算结果

4. 修改 `calculateInitialAmount` 方法：
   - 返回折扣ID和折扣率信息

5. 修改 `createOrder` 方法：
   - 设置订单的折扣相关字段

6. 修改 `calculateExtendAmount` 方法：
   - 调用新的折扣计算逻辑

7. 新增 `calculateExtendDiscount` 方法：
   - 计算续费时的折扣信息
   - 如果有新折扣，计算累计平均折扣率

8. 新增 `calculateAccumulatedDiscountRate` 方法：
   - 计算累计平均折扣率
   - 公式：`平均折扣率 = (当前平均折扣率 × 当前次数 + 新折扣率) / (当前次数 + 1)`
   - 示例：第一次7折(0.7)，第二次8折(0.8)，则平均折扣率为 (0.7 + 0.8) / 2 = 0.75

9. 修改 `updateOrderForExtend` 方法：
   - 更新订单的折扣字段
   - 累加折扣ID到JSON数组

10. 删除 `calculateExtendedFinalAmount` 方法（功能已合并到新方法中）

## 业务逻辑说明

### 平均折扣率计算
- 当用户多次选择活动折扣时，系统会记录每次的折扣率
- 最终的折扣价格为：`原价 × 平均折扣率`
- 平均折扣率 = 所有折扣率的算术平均值

### 示例场景
1. 用户开始计时，选择7折活动折扣
   - `discount_ids`: `["discount1"]`
   - `discount_count`: 1
   - `average_discount_rate`: 0.7000
   - 最终价格 = 原价 × 0.7

2. 用户续费，选择8折活动折扣
   - `discount_ids`: `["discount1", "discount2"]`
   - `discount_count`: 2
   - `average_discount_rate`: (0.7 + 0.8) / 2 = 0.7500
   - 最终价格 = 原价 × 0.75
